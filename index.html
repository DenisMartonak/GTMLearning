1 Set up your container

This page already contains the standard GTM snippet in the <head> and a <noscript> fallback right after <body>. To make it live with your account:

  1. Create a container at tagmanager.google.com and copy your ID (looks like GTM-ABC1234).
  2. Open index.html and replace GTM-NBJGCQTK in both places.
  3. In GTM, click Preview, enter this page's URL, and watch events arrive in Tag Assistant.
No container yet? Everything below still works — the live dataLayer inspector shows exactly what GTM would receive, so you can learn the data model before wiring up tags.

5 Testing checklist & cheat sheet

How to verify in GTM

  1. In GTM, click Preview (top right).
  2. Enter this page's URL → Connect.
  3. Tag Assistant opens alongside the page.
  4. Fire an event here; watch it appear in the left timeline.
  5. Click the event to inspect Variables, Data Layer, and which Tags Fired.

Push it yourself (console)

window.dataLayer.push({
  event: 'my_test',
  value: 42,
  page_section: 'hero'
});

Open DevTools console and paste — it'll show in the inspector above and in GTM Preview.
